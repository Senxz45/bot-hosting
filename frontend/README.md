# BotHost Frontend

## Features

✨ **Dark/Light Mode** - Theme toggle
📊 **Analytics Dashboard** - Real-time charts
🤖 **Bot Management** - Deploy and manage bots
🚀 **Deploy Options** - Docker, GitHub, Manual
⚙️ **Settings** - User preferences and security

## Project Structure

```
frontend/
├── src/
│   ├── components/    # Reusable components
│   ├── pages/        # Page components
│   ├── stores/       # Zustand stores
│   ├── index.css     # Tailwind styles
│   ├── App.jsx       # Main App
│   └── main.jsx      # Entry point
├── index.html
└── package.json
```

## Installation

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```
